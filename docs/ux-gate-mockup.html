설계 의도 — 게이트 통과 후 다시는 마주치지 않는다. 모델 상태는 푸터의 작은 점 하나로 minimization. 클릭하면 모델 정보·재다운로드·삭제 옵션 팝오버. 평소엔 인지 부하를 주지 않는 조용한 안심 신호.

안티-다크패턴 체크리스트

  • ✓ 강제임을 숨기는 작은 회색 "건너뛰기" 버튼 — 없음
  • ✓ 사용자 동의 없는 자동 다운로드 — 없음 (명시 클릭 후 시작)
  • ✓ "지금 안 설치하면 위험합니다" 압박 — 없음 (가치 제시만)
  • ✓ 진행률만 빙빙 도는 화면 — 없음 (이어설치하기 안심 메시지 동반)
  • ✓ 완료 후 빈 화면 — 없음 (즉시 첫 사용 + 안전 데모)
  • ✓ 매 진입마다 "AI 활성" 배너 — 없음 (푸터 점 하나로)

화면 전이 흐름

  1. 설치 직후 → 화면 1 (게이트)
    사이드패널 첫 진입 시 모델 미설치 → 입력 UI 전부 disable, 게이트만 표시.
  2. "AI 모델 설치하기" 클릭 → 화면 2 (진행)
    진행 중 사이드패널 닫아도 설치는 background에서 계속.
  3. 다운로드 완료 → 화면 3 (준비 완료)
    3초 후 자동 dismiss하고 화면 5로 슬라이드 — 또는 사용자가 액션 카드 클릭.
  4. 실패/네트워크 오류 → 화면 4
    이어설치하기 가능. 푸터 상태가 빨간 점.
  5. 이후 평상시 → 화면 5
    게이트는 다시 마주치지 않음. 모델 업그레이드 시에도 백그라운드 처리.
