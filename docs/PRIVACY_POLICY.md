# 오렌지 필터 개인정보 처리방침 (Privacy Policy)

**서비스명**: 오렌지 필터 (Chrome Extension)
**버전**: 0.0.1
**시행일**: 2026-04-28
**문의**: pengdo@myorange.io
**관련 문서**: [위협 모델](./THREAT_MODEL.md)

---

## 1. 핵심 약속 (TL;DR)

1. **모든 처리는 이 PC 안에서 이뤄집니다.** 사용자가 입력한 텍스트, 업로드한 파일, 마스킹 결과 — 어떠한 컨텐츠도 외부 서버로 전송되지 않습니다.
2. **수집·저장하지 않습니다.** 우리는 사용자의 paste 컨텐츠나 파일을 우리 서버에 저장하지 않습니다 — 우리에게 서버가 없습니다.
3. **모델 다운로드만 외부 통신입니다.** 첫 사용 시 HuggingFace에서 NER 모델 가중치를 1회 다운로드합니다. 이후 오프라인에서 동작합니다.
4. **텔레메트리·분석 0건.** 사용 통계, 에러 리포트, 사용자 ID 추적 — 모두 없습니다.

---

## 2. 우리가 처리하는 정보

### 2.1 사용자가 직접 제공하는 컨텐츠

| 데이터 | 처리 목적 | 처리 위치 | 보관 기간 |
|--------|-----------|-----------|-----------|
| **paste 텍스트** | LLM 입력 전 PII 자동 탐지·마스킹 | 사용자 PC (메모리) | paste 이벤트 처리 직후 GC. 디스크 비저장 |
| **업로드 파일 컨텐츠** | 파일 내 PII 탐지·마스킹 후 결과물 다운로드 | 사용자 PC (메모리 + IndexedDB 임시 큐) | 큐에서 제거 시 즉시 삭제. 사이드패널 닫으면 휘발 |
| **마스킹 결과물** | 사용자가 다운로드 (`<a download>`) | 사용자 PC (디스크) | 사용자가 직접 관리 |

### 2.2 자동으로 저장되는 설정

| 데이터 | 저장 위치 | 처리 목적 |
|--------|-----------|-----------|
| 카테고리별 마스킹 활성화/모드 | `chrome.storage.local` (디바이스 내부) | 사용자 선호 영속화 |
| 화이트리스트 도메인 | `chrome.storage.local` | 특정 사이트에서 모달 비활성화 |
| 사용자 모드 (default/multilingual/precision) | `chrome.storage.local` | 모델 라우팅 결정 |

`chrome.storage.local`의 데이터는 **확장 제거 시 자동 삭제**되며, 외부에 송신되지 않습니다.

### 2.3 외부 통신 (단 1건)

**모델 가중치 다운로드**:
- **수신지**: `https://huggingface.co/Xenova/bert-base-NER/...` (또는 v1.1+에서 KoELECTRA 등)
- **목적**: 온디바이스 NER 추론을 위한 모델 가중치 1회 다운로드 (~30MB int8 quantized)
- **빈도**: 첫 사용 시 1회. 이후 IndexedDB 캐시에서 로드 (오프라인 동작)
- **송신 데이터**: 표준 HTTP 요청 헤더만 (User-Agent, Referer 등 브라우저 기본값). **사용자 컨텐츠 송신 없음**.
- **HuggingFace 개인정보처리방침**: https://huggingface.co/privacy (외부 링크)

**그 외 외부 통신**: 0건. 광고, 분석 SDK, 에러 리포트, 사용 통계 — 모두 부재합니다.

## 3. 우리가 하지 않는 일 (Negative Disclosures)

명시적으로 약속드립니다 — 우리는 다음을 **하지 않습니다**:

- ❌ 사용자 paste 텍스트를 우리 서버로 전송하지 않습니다
- ❌ 업로드 파일을 외부에 업로드하지 않습니다
- ❌ 텔레메트리, 사용 통계, 분석 이벤트를 수집하지 않습니다
- ❌ 사용자 ID, 디바이스 ID, 광고 ID를 생성하거나 추적하지 않습니다
- ❌ 쿠키를 설정하지 않습니다
- ❌ 제3자 SDK (Google Analytics, Sentry, Amplitude 등)를 포함하지 않습니다
- ❌ 사용자 데이터를 광고에 사용하지 않습니다
- ❌ 사용자 데이터를 머신러닝 학습에 사용하지 않습니다
- ❌ 사용자 데이터를 제3자에게 판매하거나 공유하지 않습니다

이 목록의 항목 중 하나라도 변경된다면, 변경 시점 **이전에** 사용자에게 명시적으로 알리고 옵트인 동의를 받습니다.

## 5. 권한 요청 (Chrome Permissions)

manifest에서 요청하는 권한과 그 사용 목적입니다.

| 권한 | 용도 |
|------|------|
| `storage` | 사용자 설정 영속화 (§2.2) |
| `sidePanel` | 파일 업로드 사이드 패널 표시 |
| `offscreen` | Service Worker 수명을 넘어 모델 메모리 유지 |
| `host_permissions: chatgpt.com, claude.ai, gemini.google.com, perplexity.ai, *.orangeimpact.kr` | paste 후킹 대상 LLM 사이트 |

`tabs`, `webRequest`, `cookies`, `history`, `bookmarks` 등 고권한은 **요구하지 않습니다**.

## 6. 어린이 사용자

본 확장은 만 14세 미만 어린이를 대상으로 하지 않습니다. 어린이의 개인정보를 의도적으로 수집·처리하지 않습니다 (또한, 어떤 사용자의 개인정보도 외부로 송신하지 않습니다 — §3).

---

## 7. 사용자 권리

본 확장은 사용자의 개인정보를 디바이스 밖으로 송신하지 않으므로, 일반적인 "정보 열람·삭제" 요청 대상 데이터가 우리 측에 존재하지 않습니다.

디바이스 내 데이터 관리:
- **설정 초기화**: 확장 옵션에서 "기본값 복원" 또는 확장 제거 → 재설치
- **모델 캐시 삭제**: Chrome → 설정 → 개인정보 → 사이트 데이터 → `chrome-extension://` 항목
- **확장 제거**: `chrome://extensions` → 제거. 모든 로컬 데이터 자동 삭제

## 8. 변경 통지

본 처방침이 변경될 경우:
- **중대한 변경** (예: 새 외부 통신 추가, 데이터 수집 시작): 변경 시점 이전에 확장 내 알림으로 고지하고 사용자 동의 절차 진행
- **사소한 변경** (예: 표현 정정, 의존성 버전 업데이트): GitHub 저장소 [CHANGELOG.md](../CHANGELOG.md) 및 본 문서 §10 변경 이력 갱신

## 9. 관할 법률 / 준거법

본 처방침은 대한민국 법률에 따라 해석됩니다. 다음 법률을 준수합니다:
- 개인정보 보호법 (PIPA)
- 정보통신망 이용촉진 및 정보보호 등에 관한 법률
- (해외 사용자) GDPR (EU), CCPA (California) — 본 확장은 데이터 송신이 없으므로 "controller / processor"에 해당하지 않으나, "데이터 최소화" 원칙은 자발적으로 준수
